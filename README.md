# Японский кабинет

## Запуск
1. Откройте `auth.html` через Live Server.
2. Войдите/зарегистрируйтесь или войдите как гость.
3. После входа автоматически открывается `index.html`.

## Что есть
- Отдельная страница авторизации
- Личный кабинет с фото профиля
- Стрик, сердца, XP, уровень, цель
- Путь прогресса в стиле учебного трека
- Грамматика, кандзи, словарь, тренировки
